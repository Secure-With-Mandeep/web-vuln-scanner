# 🔍 Web Application Vulnerability Scanner

A Python-based DAST (Dynamic Application Security Testing) tool that
automatically scans web applications for common vulnerabilities.

## Status
🚧 Work in progress

## Tech Stack
- Python 3
- Flask
- Requests
- BeautifulSoup
- DVWA (test target)

## Vulnerabilities Detected
- SQL Injection
- Cross Site Scripting (XSS)
- Sensitive file exposure
- Missing security headers
- Directory traversal
